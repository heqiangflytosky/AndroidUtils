# AndroidUtils
实现一些Java和Android中常用的工具类

## AESUtil
 - Java实现AES加密
 - Java实现AES解密

## AppUtil
一些针对Android App的常用操作
 - 获得当前应用程序的名称
 - 获得当前应用程序的版本号
 - 通过包名启动App
 - 获取系统已经安装应用信息

## ColorUtil
 - 对两种颜色进行混合
 - 比较两种颜色

## DensityUtil
 - dp 转成为 px
 - px(像素) 转成为 dp

## FileUtil
 - 对文件的复制
 - 对文件的删除
 - 获取文件MD5功能

## MD5Util
 - 获取字符串的MD5值

## NetWorkUtil
 - 检测设备当前网络是否可用
 - 检测当前网络类型(wifi、2g、3g、4g、off、unknown)

## StringUtil
 - 对字符串异或加密
